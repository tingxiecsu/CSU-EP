# CUREI
This is the code repo for the paper Contrastive alignment of simulated and experimental electron ionization mass spectra for high-fidelity library matching. We developed a method named CUREI to bridge simulated and experimental spectra using Self-supervised pre-training and contrastive fine-tuning, enabling robust cross-domain alignment and accurate compound identification.
